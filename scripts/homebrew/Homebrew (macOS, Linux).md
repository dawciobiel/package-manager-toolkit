# Homebrew (macOS/Linux)

```shell
# Instalacja pakietu
brew install PACKAGE_NAME

# Wyszukiwanie pakietu
brew search PACKAGE_NAME

# Usunięcie pakietu
brew uninstall PACKAGE_NAME

# Aktualizacja pakietu
brew upgrade PACKAGE_NAME  # lub pełna aktualizacja: brew upgrade

# Odświeżenie listy pakietów
brew update

# Obsługa "supervisor" (brak bezpośredniej opcji, ale można sprawdzić):
brew outdated
```
